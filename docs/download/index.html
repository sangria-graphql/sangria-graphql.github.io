Downloads

The latest version of the library is 4.0.0. You can view and download the sources directly from the GitHub repo. If you want to download jars, then please do so directly from the maven central.

sangria

GitHub
sangria
Latest version
4.0.0
Maven central
for Scala 2.11, 2.12

You can use following dependency in your SBT build:

libraryDependencies += "org.sangria-graphql" %% "sangria" % "4.0.0"
